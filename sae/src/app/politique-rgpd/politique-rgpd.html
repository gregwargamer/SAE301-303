Politique de Protection des Données Personnelles

Responsable du traitement

Eishi, 17 Rue Jablinot, 77100 Meaux, France, est responsable du traitement de vos données. Contact : contact@eishi.fr.

Données collectées et finalités

Nous collectons vos noms, prénoms, adresses, emails et numéros de téléphone uniquement lorsque vous les fournissez volontairement :

  • • Pour gérer vos commandes et maintenir votre session de navigation (ex. : panier, connexion), sur la base de l'exécution d'un contrat (art. 6.1.b RGPD).
  • • Pour vous envoyer une newsletter, uniquement si vous donnez votre accord préalable, sur la base de votre consentement (art. 6.1.a RGPD).
  • • Pour des raisons commerciales (ex. : analyser vos commandes pour améliorer notre offre, vous envoyer des offres personnalisées), uniquement avec votre consentement préalable via un popup (art. 6.1.a RGPD).

Formulaires (inscription, contact, newsletter)

Chaque formulaire inclut une notice sous le formulaire avec :

  • • Une case obligatoire (non cochée par défaut) : « J'accepte que mes données soient utilisées pour gérer mes commandes » + lien vers cette politique de protection des données.
  • • Une case optionnelle (non cochée par défaut) : « J'accepte de recevoir la newsletter » + lien vers cette politique de protection des données. Les préférences sont modifiables dans « Mon compte » ou via le lien de désabonnement.

Cookies

  • • Cookies de session : temporaires, nécessaires au panier et à la connexion, effacés à la fermeture du navigateur. Exemptés de consentement (CNIL).
  • • Cookies commerciaux : activés uniquement si vous acceptez via le popup (marketing, analyse, personnalisation). Refus possible à tout moment.

Durées de conservation

  • • Commandes/facturation : 10 ans (Code de commerce, art. L123-22).
  • • Autres données : 2 ans après dernière interaction ou désabonnement.
  • • Données excédentaires supprimées ou anonymisées.

Destinataires des données

Accès limité aux équipes internes d'Eishi (gestion commandes, livraison).

Transferts hors EEE

Aucun.

Vos droits

Vous pouvez :

  • • Accéder : consulter vos données dans « Mon compte ».
  • • Rectifier : modifier vos informations dans « Mon compte ».
  • • Supprimer : demander la suppression dans « Mon compte » ou via « Contact ».
  • • Limiter : signaler une limitation via « Contact ».
  • • Opposition : vous opposer aux cookies commerciaux dans « Mon compte ».
  • • Retirer le consentement : via le popup, le lien de désabonnement, ou dans « Mon compte ».
  • • Portabilité : exporter vos données depuis « Mon compte ».
  • • Sort après décès : définir vos volontés via « Contact ».

Exercez ces droits via la section « Mon compte » (si fonctionnalités disponibles), le formulaire « Contact » sur notre site, ou par email à contact@eishi.fr (veuillez joindre une preuve d'identité si nécessaire).
